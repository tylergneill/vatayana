PP.0. maṅgala

1

ātmakhyātir asatkhyātir akhyātiḥ khyātir anyathā / parīkṣakāṇāṃ vibhrāntau vivādāt sā vivicyate // 1 //

PP.I. ātmakhyāti

2

asac cakāsti na vyomakusumaṃ na ........... / arthaḥ prakāśate 'to dhīs tadākāreti kecana // 2 //

3

asaṃbhavi ca yāvac ca tāvat saṃparihīyatām / saṃvedyākārasaṃtyāgo na bhrāntitve 'vakalpate // 3 //

4

PP.II.A bauddha asatkhyāti

5

anyo bahir bhāsamāno nāntar arhati dhāraṇam / asattaiva varaṃ tasya bhrāntitvānuguṇā hi sā // 5 //

6

nāntarvartitayā bhrāntir asattvena tu seṣyate / akalpayitvāntas tasmād asattāṃ khalv acīkḷpat // 6 //

7

nāntarvartibahirbhāso bhrāntitvānupapattitaḥ / asattvenaiva tatkḷptau na pramāṇavatī hi sā // 7 //

8

yathāvabhāsamānasya kalpyāsattā niyogataḥ / antarbhāve 'pi ca bahirbhāve bhrāntir na yujyate // 8 //

9

akhyāter aviśeṣeṇa syāt suṣupte 'pi vibhramaḥ / akhyātiḥ khalu tatrāpi na cākhyātir viśiṣyate // 9 //

10

upālambhasya na padam asato 'tra prakāśanam / ata eva yato bhrāntiḥ samyakkhyātau tathā bhavet // 10 //

11

14

15

16

17

18

19

20

22

23

doṣair avikṛtaṃ svasthaṃ praṇidhānādisaṃskṛtam / na kāryavaj jāgarāyām aho nayavidāṃ paraḥ // 23 //

24

doṣakṣatiḥ kāryaśaktihānirūpā ca vidyate / atirekaś ca kāryasyety aho nyāyavivekitā // 24 //

25

adhyārope bhaved gauṇī vāhīke gomatir yathā / na saṃvidānuguṇyaṃ syān na vivekamatir yadi // 25 //

26

akhyātir api saṃvittiṃ naivānveti yato matiḥ / sāmānādhikaraṇyena rūpyam etad iti sthitā // 26 //

27

tasmād vibhrama evāyam iti yukto viniścayaḥ / na saṃvidanusāreṇa nimittaṃ tasya yujyate // 27 //

PP.II.B vedānta asatkhyāti

28

ato 'nirvacanīyatvaṃ varaṃ brahmavido viduḥ / avidyāyā avidyātvam anyathā parihīyate // 28 //

29

sattve na mithyā śūnyatve durnirūpaṃ prakāśanam / sadasadbhyām anirvācyāṃ tām avidyāṃ pracakṣate // 29 //

30

31

32

33

asphuṭagrahaṇe kāmam mā bhasi sphuṭam ātmanā / avidyamāne tv adhyasye vaiśvarūpyaṃ vṛthākṛtam // 33 //

34

citau vicitrākārāyāṃ prapañcātmatayaiva hi / anirmokṣas tathā ca syād athavānityatāpatet // 34 //

35

anekākāravibhrāntau gandharvanagarādiṣu / ākārā vyaktam ekasyā dhiyo 'satyāś cakāsati // 35 //

36

na bhūtaṃ cetaso rūpaṃ nādhyāropāsphuṭagrahau / vibhrameṣu vivartatvam ato brahmavidāṃ matam // 36 //
