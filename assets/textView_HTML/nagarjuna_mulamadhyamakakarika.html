vijñānaṃ saṃniviśate saṃskārapratyayaṃ gatau / saṃniviṣṭe 'tha vijñāne nāmarūpaṃ niṣicyate //

pañca skandhāḥ sa ca bhavo bhavāj jātiḥ pravartate / jarāmaraṇaduḥkhādi śokāḥ saparidevanāḥ //

27.1 (MMK_27.1_27.4)

abhūm atītam adhvānaṃ nābhūvam iti dṛṣṭayaḥ / yās tāḥ śāśvatalokādyāḥ pūrvāntaṃ samupāśritāḥ //

27.2

dṛṣṭayo na bhaviṣyāmi kim anyo 'nāgate 'dhvani / bhaviṣyāmīti cāntādyā aparāntaṃ samāśritāḥ //

27.3

abhūm atītam adhvānam ity etan nopapadyate / yo hi janmasu pūrveṣu sa eva na bhavaty ayam //

27.4

sa evātmeti tu bhaved upādānaṃ viśiṣyate / upādānavinirmukta ātmā te katamaḥ punaḥ //

27.5 (MMK_27.5_27.8)

upādānavinirmukto nāsty ātmeti kṛte sati / syād upādānam evātmā nāsti cātmeti vaḥ punaḥ //

27.6

na copādānam evātmā vyeti tat samudeti ca / kathaṃ hi nāmopādānam upādātā bhaviṣyati //

27.7

anyaḥ punar upādānād ātmā naivopapadyate / gṛhyeta hy anupādāno yady anyo na ca gṛhyate //

27.8

evaṃ nānya upādānān na copādānam eva saḥ / ātmā nāsty anupādāno nāpi nāsty eṣa niścayaḥ //

27.9 (MMK_27.9_27.12)

nābhūm atītam adhvānam ity etan nopapadyate / yo hi janmasu pūrveṣu tato 'nyo na bhavaty ayam //

27.10

yadi hy ayaṃ bhaved anyaḥ pratyākhyāyāpi taṃ bhavet / tathaiva ca sa saṃtiṣṭhet tatra jāyeta cāmṛtaḥ //

27.11

ucchedaḥ karmaṇāṃ nāśaḥ kṛtam anyena karma ca / pratisaṃvedayed anya evam ādi prasajyate //

27.12

nāpy abhūtvā samudbhūto doṣo hy atra prasajyate / kṛtako vā bhaved ātmā saṃbhūto vāpy ahetukaḥ //

27.13 (MMK_27.13_27.16)

evaṃ dṛṣṭir atīte yā nābhūm aham abhūm aham / ubhayaṃ nobhayaṃ ceti naiṣā samupapadyate //

27.14

adhvany anāgate kiṃ nu bhaviṣyāmīti darśanam / na bhaviṣyāmi cety etad atītenādhvanā samam //

27.15

sa devaḥ sa manuṣyaś ced evaṃ bhavati śāśvatam / anutpannaś ca devaḥ syāj jāyate na hi śāśvatam //

27.16

devād anyo manuṣyaś ced aśāśvatam ato bhavet / devād anyo manuṣyaś cet saṃtatir nopapadyate //

27.17 (MMK_27.17_27.19)

divyo yady ekadeśaḥ syād ekadeśaś ca mānuṣaḥ / aśāśvataṃ śāśvataṃ ca bhavet tac ca na yujyate //

27.18

aśāśvataṃ śāśvataṃ ca prasiddham ubhayaṃ yadi / siddhe na śāśvataṃ kāmaṃ naivāśāśvatam ity api //

27.19

kutaścid āgataḥ kaścit kiṃcid gacchet punaḥ kva cit / yadi tasmād anādis tu saṃsāraḥ syān na cāsti saḥ //

nāsti cec chāśvataḥ kaścit ko bhaviṣyaty aśāśvataḥ / śāśvato 'śāśvataś cāpi dvābhyām ābhyāṃ tiraskṛtaḥ //

27.21

antavān yadi lokaḥ syāt paralokaḥ kathaṃ bhavet / athāpy anantavāl lokaḥ paralokaḥ kathaṃ bhavet //

27.22

skandhānām eṣa saṃtāno yasmād dīpārciṣām iva / tasmān nānantavattvaṃ ca nāntavattvaṃ ca yujyate //

27.23 (MMK_27.23_27.25)

pūrve yadi ca bhajyerann utpadyeran na cāpy amī / skandhāḥ skandhān pratītyemān atha loko 'ntavān bhavet //

27.24

27.25
