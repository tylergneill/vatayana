sambandhaparīkṣā

01

pāratantryaṃ hi sambandhaḥ siddhe kā paratantratā / tasmāt sarvasya bhāvasya sambandho nāsti tattvataḥ //

02

rūpaśleṣo hi sambandho dvitve sa ca kathaṃ bhavet / tasmāt prakṛtibhinnānāṃ sambandho nāsti tattvataḥ //

03

parāpekṣā hi sambandhaḥ so 'san katham apekṣate / saṃś ca sarvanirāśaṃso bhāvaḥ katham apekṣate //

04

dvayor ekābhisambandhāt sambandho yadi taddvayoḥ / kaḥ sambandho 'navasthā ca na sambandhamatis tathā //

05

tau ca bhāvau tadanyaś ca sarve te svātmani sthitāḥ / ity amiśrāḥ svayaṃ bhāvās tān miśrayati kalpanā //

06

tām eva cānurundhānaiḥ kriyākārakavācinaḥ / bhāvabhedapratītyarthaṃ saṃyojyante 'bhidhāyakāḥ //

07

kāryakāraṇabhāvo 'pi tayor asahabhāvataḥ / prasiddhyati kathaṃ dviṣṭho 'dviṣṭhe sambandhatā katham //

08

krameṇa bhāva ekatra vartamāṇo 'nyanispṛhaḥ / tadabhāve 'pi tadbhāvāt sambandho naikavṛttimān //

09

yady apekṣya tayor ekam anyatrāsau pravartate / upakārī hy apekṣyaḥ syāt kathaṃ copakaroty asan //

10

yady ekārthābhisambandhāt kāryakāraṇatā tayoḥ / prāptā dvitvādisambandhāt savyetaraviśāṇayoḥ //

11

dviṣṭho hi kaścit sambandho nāto 'nyat tasya lakṣaṇam / bhāvābhāvopadhir yogaḥ kāryakāraṇatā yadi //

12

yogopādhī na tāv eva kāryakāraṇatātra kim / bhedāc cen nanv ayaṃ śabdo niyoktāraṃ samāśritaḥ //

13

paśyann ekam adṛṣṭasya darśane tadadarśane / apaśyat kāryam anveti vinā vyākhyātṛbhir janaḥ //

14

darśanādarśane muktvā kāryabuddher asambhavāt / kāryādiśrutir apy atra lāghavārthaṃ niveśitā //

15

tadbhāvābhāvāt tatkāryagatir yāpy anuvarṇyate / saṃketaviṣayākhyā sā sāsnāder gogatir yathā //

16

bhāve bhāvini tadbhāvo bhāva eva ca bhāvitā / prasiddhe hetuphalate pratyakṣānupalambhataḥ //

17

etāvanmātratattvārthāḥ kāryakāraṇagocarāḥ / vikalpā darśayanty arthān mithyārthā ghaṭitān iva //

18

bhinne kā ghaṭanābhinne kāryakāraṇatāpi kā / bhāve hy anyasya viśliṣṭau śliṣṭau syātāṃ kathaṃ ca tau //

19

saṃyogisamavāyyādi sarvam etena cintitam / anyonyānupakārāc ca na sambandhī ca tādṛśaḥ //

20

janane 'pi hi kāryasya kenacit samavāyinā / samavāyī tadā nāsau na tato 'tiprasaṅgataḥ //

21

tayor anupakāre 'pi samavāye paratra vā / sambandho yadi viśvaṃ syāt samavāyi parasparam //

22

saṃyogajanane 'pīṣṭau tataḥ saṃyoginau na tau / karmādiyogitāpatteḥ sthiteś ca prativarṇitā //

23

saṃyogādyāśraye yogyam ayogyaṃ tac ca jāyate / nityaṃ yogyasvabhāvasya tadvaikalyavirodhataḥ //

24 (SAS_24_27)

iti tadyogyatāvācyaḥ svabhāvo 'sya nirucyatām / vibhāgayogagatibhiḥ kim anyair gamanādibhiḥ //

25

teṣu satsv api tasyeti sambandhasyāprasiddhitaḥ / yuktaḥ svabhāvabhedo 'yam tat pratikṣaṇajanmanām //
